Linux 配置 SSH 服务实现远程远程访问_使用 ssh 远程控制 linux_liguangxian2018 的博客 - CSDN 博客

  1. 查询是否安装 SSH.
rpm -pa |grep ssh
  1. 如果没有安装 rmp:
sudo apt-get install rpm          #ubuntu,debian
yum -y instal rpm                 #centos,redhat
  1. 安装 SSH
sudo apt-get install ssh
or
yum -y install openssh
  1. 启动服务:
service sshd start
or
/bin/systemctl restart sshd.service
or
/etc/init.d/sshd start
  1. 配置端口:
vim /etc/ssh/sshd_config
  1. 将 port 前面的 #删除, 也可以更改其它端口.

  1. 允许 root 用户远程登录.

  1. 查看服务是否启动:
ps -e | grep ssh

  1. 远程登录:
ssh name@192.168.0.128
